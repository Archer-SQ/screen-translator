One-Key Translation

Press a single hotkey combination to capture your screen, detect text, translate, and overlay — all in one seamless step.

🎨

Pixel-Perfect Overlay

Canvas-based rendering matches the original font size, background color, and position. Translations look native.

🔒

Privacy First

Everything runs locally on your Mac. Screenshots never leave your device. API keys are stored only in your local config.

🎯

Smart Detection

Combines macOS Vision OCR with Accessibility API for precise text detection and positioning across any application.

💾

Translation Cache

Save translations you want to keep with Shift+S. Cached results display instantly without re-translating.

Fully Configurable

Customize hotkeys, choose your translation provider, set your target language — all from a clean settings interface.
