为什么选择 Screen Translator

专为浏览器翻译不了的场景而生

全屏一键翻译

按 Shift+Z+X，整屏的 UI、菜单、内容一键翻译，译文直接覆盖原文位置。

选区翻译

按 Shift+Z+C 框选任意区域，只翻译你关心的部分。结果浮层可自由拖拽缩放。

冻结截图

按下快捷键瞬间画面冻结，动态视频、游戏、动画也能精确框选，所见即所得。

像素级覆盖

Canvas 直接绘制，自动匹配字号和背景色，看起来像原生汉化。

多屏支持

自动检测光标所在的屏幕，把鼠标放到哪个屏幕就翻译哪个屏幕。

浮层随心

任意位置可拖拽移动，8 方向边缘缩放，触控板双指捏合，双击关闭。
