简洁而强大

零配置开始，开箱即用

一键翻译

按下快捷键，自动完成截屏、文字识别、翻译、覆盖显示，全程无需切换窗口。

像素级覆盖

Canvas 直接绘制，自动匹配原始字号、背景色和位置，译文如同原生显示。

隐私优先

所有处理在本地完成，截图不会上传。API 密钥仅存储在本地配置文件中。

精准识别

结合 macOS Vision OCR 与 Accessibility API，精准检测文字位置，适配各种界面。

翻译缓存

按 Shift+S 保存翻译，相同内容瞬间显示，无需重新翻译。

自由配置

自定义快捷键、翻译服务、目标语言，支持中英双语设置界面。
