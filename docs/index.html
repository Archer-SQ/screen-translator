为什么选择 Screen Translator

专为浏览器翻译不了的场景而生

任何应用都能翻译

不限于浏览器 — 开发工具、设计软件、游戏界面，只要屏幕上有文字就能翻译。

像素级原位覆盖

译文直接覆盖原文位置，自动匹配字号和背景色，看起来像原生汉化。

隐私优先

截图在本地处理，不上传任何服务器。API 密钥仅存储在你的电脑上。

OCR + 无障碍双引擎

macOS Vision OCR 识别文字，Accessibility API 精确定位，两者结合精度更高。

一键缓存

常用界面按 Shift+S 缓存，下次秒显，不用重新等待翻译。

免费开箱即用

内置 Google 免费翻译，也支持 OpenAI、Claude、DeepL、Ollama 等付费/本地方案。
